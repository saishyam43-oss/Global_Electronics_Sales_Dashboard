# Global Electronics Sales Dashboard

An end-to-end retail sales analytics project built using SQL and Power BI.  
This project demonstrates advanced data cleaning, data modeling, and DAX-driven insights for business performance tracking across products, customers, time, and regions.

---

**Project Highlights**
- Cleaned 5 messy source datasets (Sales, Products, Customers, Stores, Exchange Rates)
- Built a robust star schema data model in Power BI
- Designed a 5-page analytical dashboard for executives and business teams
- Quantified data reliability (99.6%) and tracked sales performance across years
